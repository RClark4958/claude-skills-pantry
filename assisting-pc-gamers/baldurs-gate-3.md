# Baldur's Gate 3 - Common Issues & Solutions

## Technical Issues

### Save Game Problems
**Symptoms:** Can't save, saves corrupted, or saves missing
**Solutions:**
1. Check storage space (needs 10GB+ free)
2. Disable cloud saves temporarily
3. Run as administrator
4. Check Documents/Larian Studios/Baldur's Gate 3 folder permissions
5. Disable antivirus scanning on save folder
6. Use different save slots (don't overwrite)

### Multiplayer Desync
**Symptoms:** Players see different things, combat out of sync
**Solutions:**
1. Host should quicksave and reload
2. All players restart game
3. Verify game files
4. Check all players on same version/patch
5. Disable mods if using any

### Crashes in Act 3
**Common in Lower City**
**Solutions:**
1. Lower graphics settings (especially in city)
2. Disable dynamic crowds
3. Fast travel instead of walking
4. Clear inventory of excess items
5. Dismiss unnecessary summons/companions

## Progression Issues

### Can't Enter Goblin Camp
**Solution:** Multiple approaches:
- Talk your way in (various dialogue options)
- Use disguise (Disguise Self spell)
- Sneak through broken wall on west side
- Fight your way in

### Stuck in Turn-Based Mode
**Solutions:**
1. Press Shift + Space to toggle
2. Check if any character still in combat
3. Long rest to reset
4. Save and reload

### Can't Find Nightsong
**Location:** Shadowfell, access through Gauntlet of Shar
**Requirements:**
- Complete all Shar trials
- Get all Umbral Gems
- Use gems on pedestal
- Take elevator down

## Quest Bugs & Solutions

### Halsin Won't Join
**Requirements:**
- Save him from goblins
- Complete Grove conflict (any resolution)
- Talk to him at camp after celebration
- Must happen before Act 2

### Karlach's Quest Broken
**Common Issues:**
- Need Infernal Iron (multiple locations)
- Dammon must survive Act 1 & 2
- Check Last Light Inn in Act 2
- Final upgrade in Act 3 (Lower City)

### Lift Not Working (Gauntlet of Shar)
**Solution:**
- Need all 4 Umbral Gems
- One from each trial (3)
- One from Yurgir or sneaking
- Place all gems before using lift

### Can't Romance Character
**Common Requirements:**
- High approval (check reactions)
- Complete personal quest
- Choose correct dialogue options
- Some lock out others
- Long rest frequently for camp scenes

## Combat Issues

### Difficulty Spike in Act 2
**Tips:**
- Level 5-7 recommended
- Get light sources for shadow curse
- Recruit all available allies
- Use Spirit Guardians/Fireball
- Position before combat starts

### Can't Beat Apostle of Myrkul
**Strategy:**
- Focus adds first
- Use radiant damage
- Spread out (avoid AoE)
- Save inspirations for death saves
- Have resurrection scrolls ready

### Raphael Fight Too Hard
**Tips:**
- Level 11+ recommended
- Steal/destroy pillars first
- Use Hope as ally
- Counterspell his big spells
- Globe of Invulnerability helps

## Build/Character Issues

### Respec Not Working
**Solution:**
- Need to pay Withers (100g per level)
- Can't change race/background
- Companions keep their special features
- Origin characters can't change class

### Multiclass Problems
**Requirements:**
- Need 13 in primary stat
- Some combos don't work well
- Extra Attack doesn't stack
- Spell slots combine weirdly

### Missing Feats
**Note:** Feats at levels 4, 8, 12
**Fighter:** Extra at 6
**Rogue:** Extra at 10

## Item Issues

### Can't Find Legendary Items
**Blood of Lathander:**
- Rosymorn Monastery
- Need Dawnmaster's Crest
- Solve weapon puzzle

**Githyanki Silver Sword:**
- Multiple methods
- Easiest: Pickpocket Voss
- Or: Complete Lae'zel's quest

**Orphic Hammer:**
- Deal with Raphael
- OR: Steal from House of Hope

### Inventory Management
**Tips:**
- Send items to camp (right-click)
- Use camp chest for storage
- Sort by weight/value
- Sell to same vendor (attitude bonus)

## Act-Specific Issues

### Act 1 Checklist
Before Mountain Pass/Underdark:
- Save Halsin
- Resolve Grove conflict
- Find Karlach + Wyll
- Get Sussur weapon
- Clear goblin leaders

### Act 2 Requirements
**Must Do:**
- Resolve Nightsong
- Save/Kill Isobel
- Deal with Thorm bosses
- Complete companion quests
- Get all allies for final battle

### Act 3 Performance
**Known Issues:**
- Lower City lag
- Long loading times
- Memory leaks after long sessions
**Workarounds:**
- Restart game every 2-3 hours
- Lower settings in city
- Disable unnecessary overlays

## Save Scumming Tips

### Honor Mode
**Cannot save scum normally**
**Workarounds:**
- Alt+F4 before death registers
- Backup saves manually
- Use withers for emergency revive

### Important Save Points
- Before major conversations
- Before entering new acts
- Before boss fights
- Before long rest (romance scenes)
- Before using legendary actions
